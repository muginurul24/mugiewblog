## Coding Conventions

### PHP
- Curly braces ALWAYS, even single-line bodies
- Constructor property promotion: `public function __construct(public Service $svc) {}`
- Explicit return types on ALL methods
- TitleCase for Enum keys: `ArticleStatus`, `UserRole`
- PHPDoc blocks > inline comments
- `#[\NoDiscard]` on methods whose return values must be consumed

### Laravel
- Use `php artisan make:` commands (NEVER manually create files)
- Service classes for business logic (`app/Services/`)
- NEVER N+1: always eager-load relationships
- Policies for ALL authorization
- Queue jobs for emails, image processing
- Redis cache tags for granular invalidation

### Livewire
- SPA routing via `Route::livewire()`
- Full-page components in `app/Livewire/Pages/`
- Reusable components in `app/Livewire/Components/`
- Layout in `app/Livewire/Layouts/`
- Use `#[Lazy]` for below-fold components
- Use `#[Island]` for independent-update components

### Filament
- Resources in `app/Filament/Resources/`
- Custom pages in `app/Filament/Pages/`
- Widgets in `app/Filament/Widgets/`
- Use `$panel->resources([])` in provider
- Forms with Section groups
- Tables with filters, bulk actions

### Frontend
- Blade for all templates (`.blade.php`)
- Tailwind utility classes (NO custom CSS unless necessary)
- Alpine.js for client-side reactivity (bundled with Livewire 4)
- Dark mode via class strategy + inline blocking script
- Icons: Blade Heroicons or Lucide

## Key Patterns

### Service Layer
```php
final class ArticleService
{
    #[\NoDiscard]
    public function getPaginated(int $page): LengthAwarePaginator
    {
        return Cache::tags(['articles'])->remember(
            "articles.paginated.{$page}",
            now()->addMinutes(30),
            fn () => Article::published()
                ->with(['author', 'category', 'tags'])
                ->withCount('comments')
                ->latest('published_at')
                ->paginate(12)
        );
    }
}
```

### Cache Invalidation
```php
// Observer-based
final class ArticleObserver
{
    public function saved(Article $article): void
    {
        Cache::tags(['articles'])->flush();
    }
}
```

### Security
- All user input validated via FormRequest or Livewire validation
- File uploads: MIME check, random filename, server-side resize
- CSRF on all non-GET routes (Laravel default)
- CSP headers via middleware
